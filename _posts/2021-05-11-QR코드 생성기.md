##### 파이썬으로 QR코드 생성하기

`qrcode` 라이브러리를 이용해 QR코드 생성하기.

```python
import qrcode
import datetime

BASE_PATH = "/Users/asadal/Downloads/"
filename = "qrcode_" + datetime.datetime.today().strftime('%Y%m%d') + ".png"
target = input("내용 또는 URL을 입력하세요 : ")
img = qrcode.make(target)
img.save(BASE_PATH + filename)
print(img.size)
```
